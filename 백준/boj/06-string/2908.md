# 2908. 상수

[problem](https://www.acmicpc.net/problem/2908)

```cpp
#include <bits/stdc++.h>
using namespace std;
#define swap(a,b) {int i=(a); (a)=(b); (b) = i;} 

int main() {
	string a,b;
	cin >> a >> b;

	swap(a[0],a[2]);
	swap(b[0],b[2]);

	int v1 = (a[0]-'0')*100 + (a[1]-'0')*10 + (a[2]-'0');
	int v2 = (b[0]-'0')*100 + (b[1]-'0')*10 + (b[2]-'0');
	cout << ((v1 < v2) ? v2 : v1) << endl;
	
	return 0;
}
```
