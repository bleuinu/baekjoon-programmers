# 1427. 소트인사이드

[problem](https://www.acmicpc.net/problem/1427)

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	string s;
	cin >> s;

	const int SIZE = s.size();
	for(int i=0; i<SIZE; ++i) {
		for(int j=0; j<SIZE;++j) {
			if(s[i] > s[j]) {
				char t = s[i];
				s[i] = s[j];
				s[j] = t;
			}
		}
	}

	for(int i=0; i<SIZE; ++i)
		cout << s[i];
	cout << endl;

	return 0;
}
```
