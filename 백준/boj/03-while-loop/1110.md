# 1110. 더하기 사이클

[problem](https://www.acmicpc.net/problem/1110)

```cpp
#include <bits/stdc++.h> 
using namespace std;

int main(void) {
  int n, cnt = 1;
  cin >> n;

  int temp = (n%10)*10 + ((n/10 + n%10))%10;
  while(n != temp) {
    temp = (temp%10)*10 + ((temp/10 + temp%10))%10;
    ++cnt;
  }
  cout << cnt << endl;

  return 0;
}
```
