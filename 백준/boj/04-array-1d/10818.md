# 10818. 최소, 최대

[problem](https://www.acmicpc.net/problem/10818)

```cpp
#include <bits/stdc++.h> 
using namespace std;

int main(void) {
	int min = 1000001;
	int max = -1000001;
	int n; 
	cin >> n;

	for(int i=0; i<n; ++i) {
		int t; 
		cin >> t;
		min = (t < min) ? t : min;
		max = (t > max) ? t : max;
	}
	cout << min << ' ' << max << endl;
	return 0;
}
```
