# 1546. 평균

[problem](https://www.acmicpc.net/problem/1546)

```cpp
#include <bits/stdc++.h> 
using namespace std;

int main() {
	int n, m=-1;
	cin >> n;
	int arr[n];
	for(int i=0; i<n; ++i) {
		cin >> arr[i];
		m = arr[i] > m ? arr[i] : m;
	}

	double sum = 0.0;
	for(int i=0; i<n; ++i) {
		sum += arr[i]*100.0 / m;
	}

	cout << sum/n << endl;
	return 0;
}
```
