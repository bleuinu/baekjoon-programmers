# 2525. 오븐 시계

[problem](https://www.acmicpc.net/problem/2525)

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int a, b, c;
	cin >> a >> b >> c;

	int mins = a*60 + b + c;
	int hour = (mins/60)%24;
	mins %= 60;

	hour = hour < 24 ? hour : 0;
	cout << hour << ' ' << mins << endl;

	return 0;
}
```
