# 2439. 별 찍기 - 2

[problem](https://www.acmicpc.net/problem/2439)

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n; cin >> n;
	for(int i=0; i<n; ++i) {
		for(int j=i+1; j<n; ++j) {
			cout << ' ';
		}
		for(int k=0; k<=i; ++k) {
			cout << '*';
		}
		cout << endl;
	}
	return 0;
}
```
